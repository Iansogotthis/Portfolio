Project 4: Destination

A travel and restaurant discovery app

Technologies used:

  1. Django: A high-level Python web framework used for rapid development and clean, pragmatic design.
  2. HTML/CSS: Standard technologies for structuring and styling web pages.
  3. JavaScript: Used for interactive elements and enhancing user experience.
  4. Bootstrap: A front-end framework for developing responsive and mobile-first websites.
  5. SQLite: A lightweight relational database management system used for storing application data.
  6. Python: The primary programming language used for backend development and business logic.
  7. Git: Version control system for tracking changes in codebase and collaboration among developers.

Features

  1. Listings: Browse a comprehensive list of restaurants, including their names, addresses, and descriptions.
  2. Photo Gallery: View photos associated with each restaurant and hotel to get a visual sense of the ambiance and cuisine.
  3. User Profiles: Create a personalized profile, add travelers, and share favorite restaurants and activities.
  4. Activity Listings: Discover various activities available in different cities, along with their descriptions.
  5. Hotel Listings: Explore a curated selection of hotels, including their names, addresses, and descriptions.
